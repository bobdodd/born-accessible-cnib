# Born Accessible CNIB

A comprehensive exploration of what it means to be a born-accessible organization, using CNIB (Canadian National Institute for the Blind) as a template.

## About This Project

This repository documents our exploration of born-accessible organizational practices, examining how accessibility principles are embedded throughout all aspects of an organization's operations.

## Repository Structure

```
born-accessible-cnib/
├── PROJECT_OVERVIEW.md           # Detailed project vision and scope
├── docs/
│   ├── staff/                    # Internal staff operations and workflows
│   ├── external-entities/        # External interactions and partnerships
│   ├── clients/                  # Client experience and service delivery
│   ├── vendors/                  # Vendor relationships and requirements
│   ├── observations/             # Key observations and insights
│   └── recommendations/          # Best practices and recommendations
```
